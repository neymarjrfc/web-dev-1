# RPG Developer Portfolio

> "From C logic to Web Magic."

A personal portfolio website designed with a Cyberpunk / RPG Gamer aesthetic. This project showcases my skills in C++, Web Development, and Competitive Programming through an interactive, gamified interface.

## Live Demo
[Click here to view](https://neymarjrfc.github.io/web-dev-1/)

## Key Features

* **RPG Character Stats:** Visual progress bars displaying proficiency in C++, Web Dev, and Typing Speed.
* **Persistent Dark Mode:** A robust light/dark toggle that saves player preference to LocalStorage (remembers your choice even after refresh).
* **Interactive UI:**
    * **Holo-Card Bio:** A 3D tilt-effect profile card using CSS perspective.
    * **Terminal Simulation:** A "command line" style intro for the About Me section.
    * **Cyber Glitch Header:** CSS animations creating a futuristic glitch effect on the username.
* **Fully Responsive:** Adapts layout for mobile (vertical stacking) and desktop (grid layout).
* **Optimized Performance:** Built with pure Vanilla JS and CSS variables (no heavy frameworks).

## Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox/Grid, CSS Variables, Animations), JavaScript (ES6).
* **Fonts:** 'Fira Code' (Monospace) and 'Poppins' (Sans-serif) via Google Fonts.
* **Icons:** CSS Shapes & SVG.
